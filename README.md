# Data Analytics Portfolio

This repository contains my data analysis projects developed during my Data Analytics training.

## Projects

### A/B Test Analysis: Subscription Conversion Optimization
Statistical analysis of an A/B test evaluating the impact of a discount message on subscription conversion.

Key result:
- Conversion increased from **6.1% to 8.9%**
- Improvement of **+2.8 percentage points**
- **p-value ≈ 0.002**, indicating statistical significance

Tools:
Python, Pandas, NumPy, statistical testing, data visualization

Project folder:
ab-test-subscription-analysis
