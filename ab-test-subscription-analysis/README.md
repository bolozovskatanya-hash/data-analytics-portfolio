# A/B Test Analysis: Subscription Conversion Optimization

## Project Overview
This project analyzes the results of an A/B test designed to evaluate the effectiveness of two subscription screen variations shown after user onboarding.

## Objective
Determine whether adding a “50% discount” message increases conversion from install to payment.

## Key Results
- Group A conversion rate: 6.1%
- Group B conversion rate: 8.9%
- Absolute increase: +2.8 percentage points
- p-value ≈ 0.002

The result indicates a statistically significant improvement.
